# 🏧 ATM Console Application (Java)

A simple **console-based ATM simulation** built using **Java and Object-Oriented Programming (OOP)** concepts.
This project demonstrates basic banking operations such as checking balance, depositing money, and withdrawing money through a command-line interface.

---

## 📌 Features

* 🔐 ATM Number & PIN authentication
* 💰 View available balance
* 💸 Withdraw money
* 💵 Deposit money
* 📟 Interactive console menu
* ❌ Exit option

---

## 🛠 Technologies Used

* Java
* OOP Concepts
* Command Line Interface (CLI)

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/ashishthakur113/ATM-Console-Project.git
```

2. Navigate to the project directory

```
cd ATM-Console-Project
```

3. Compile the Java files

```
javac src/ATMProject/*.java
```

4. Run the program

```
java src.ATMProject.MainClass
```
